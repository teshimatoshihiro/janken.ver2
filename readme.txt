# クイズ（医療とジーズ）

## クイズゲーム

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方
  医療とジーズ関係についての4択クイズです。（1問12秒以内で回答）
  正解すると正解音と「新川優愛」ちゃんの写真が背景に表示されます。不正解すると「新川優愛ちゃんが婚約した時のお相手とのツーショット写真が何枚も背景に表示されます）

  - を見た人がわかるように書こう！

## 工夫した点
今回１つの配列に問題文も４つの回答も入れて、その配列の1番が正解なのですが、問題文の順番はシャッフルして出題してます。


## 苦戦した点
正解と不正解時に「優愛ちゃん写真」と「残念写真」を出すのが難しかった。
その際の効果音は入れる事ができたが、ゲーム中のバックミュージックは入れたが再現できていません。
